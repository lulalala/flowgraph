# flowgraph

Generate a SQLite database containing a call graph for your Ruby code.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'flowgraph'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install flowgraph
